# IT1030SEF_Project

Simple Lost and Found website for students.

## Run the project

1. Install dependencies:
	`npm install`
2. Start server:
	`npm start`
3. Open:
	`http://localhost:3000`

## How to use

1. Open the website in your browser.
2. Use "I Lost Something" or "I Found Something" to submit a form.
3. Open Lost/Found Item Board to view posts.
4. Use search to filter posts.
5. Click "Answer Question to View Contact" and answer correctly.

## Data storage

- Posts are saved to SQLite database file: `backend/database.db`.
- Data persists across browser refresh and server restart.
