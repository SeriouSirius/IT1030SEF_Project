The use of knowledge covered in our lectures and tutorial: 
- Functions
- Objects
- Array
- Inputs
- Looping
- Selection
- etc.

UIUX Design:
- The website uses a dark background and light text for a clean visual style.
- Buttons and textboxes are the main interactive components.
- The page layout is simple so users can quickly submit or search for items.
- CSS is used to keep the interface consistent and readable.

Flow of using:
 When users found/lost an item, fill out the form in website
 The item will auto posted to the corresponding page
 The owner/finder search the item 
 Answers questions about item’s details
 Get the finder/owner’s contact
 Get/give back the item lost/found
